| **name** | **String** | The name of the user |  |
| **email** | **String** | The email address of the user |  |
| **role** | **String** | &#x60;owner&#x60; or &#x60;reader&#x60; |  |
| **added_at** | **Integer** | The Unix timestamp (in seconds) of when the user was added. |  |
